# LockeBox Downloads

Release builds are hosted as GitHub Release assets:

`https://github.com/heavyc-dev/_LockeBox_Releases/releases/download/release/LockeBox-Windows.exe`

Android ships through the Google Play Store:

`https://play.google.com/store/apps/details?id=com.heavyc.lockebox`

iOS ships through Apple TestFlight:

`https://testflight.apple.com/join/KXYPmBD3`

The public site buttons route through:

`/lockebox/downloads/windows/`

`/lockebox/downloads/android/`

`/lockebox/download/` is the public download page. `/lockebox/beta/` redirects there; `/lockebox/alpha/` redirects to home.
