PC Setup & Install

Connect your PC in a few steps.

node.dev links your iPhone to a Windows PC over Tailscale — a private, encrypted network. The app walks you through this during onboarding; this page is the full reference.

Windows PCThe machine you code on
iPhoneRunning node.dev
TailscaleFree account, both devices

Step 1

Install Tailscale

Install Tailscale on both the Windows PC and the iPhone, then sign in to the same tailnet on each. Tailscale is a zero-config encrypted mesh network — your devices talk directly and privately, and your PC is never exposed to the public internet.

Note your PC's MagicDNS name (e.g. rog-pc) or Tailscale IP (100.x.y.z). The setup script in Step 2 also prints this for you.

If you use custom ACLs: allow your phone to reach the PC on ports 22 (SSH) and 9922 (key push). A restrictive ACL covering only port 22 will block the key-delivery step.

Step 2

Run the PC setup script

During onboarding, node.dev gives you a one-line command to paste into PowerShell. It pops a UAC prompt and relaunches itself as Administrator. You can also download the full script and review it first:

setup.ps1 The exact script node.dev runs on your PC. Read it before running.
Download setup.ps1

What the script does

It is transparent and safe to re-run. In order, it:

  1. Installs and starts the Windows OpenSSH Server
  2. Restricts SSH to Tailscale addresses (IPv4 + IPv6) via Windows Firewall
  3. Hardens sshd_config for key-only auth — no passwords
  4. Prints the address to type into the app, then waits to receive your phone's public key automatically
  5. Verifies sshd is running

Keep the PowerShell window open. The script blocks at step 4 while it waits for the key from your phone. It times out after 10 minutes — if that happens, just re-run it and push again.

The script never truncates existing keys and only appends yours if it's not already present. It requires Administrator because installing the SSH server and editing firewall rules need elevation.

Step 3

Pair your key in node.dev

The app generates your device key automatically and stores the private key in the iPhone's hardware-backed Keychain — it never leaves the device. Enter the PC's Tailscale hostname or IP (the address the script printed) and tap Push key to PC.

The app sends the public key over Tailscale to the waiting listener, and the PC answers with its username and hostname so the next step is pre-filled. Optionally tap Verify connection to open a real SSH session and pin the host fingerprint.

If the push fails, copy the public key from the app and paste it manually into the authorized_keys file the script created.

Step 4

Add your first host and connect

The PC's hostname and Windows username are pre-filled from the pairing step. Confirm port 22 and save. SSH sessions land in PowerShell by default — you're connected.

Optional

WSL and tmux

WSL gives you a full Linux shell; tmux keeps sessions alive across disconnects. Run these after the steps above.

Install WSL with Ubuntu (requires a reboot)

wsl --install -d Ubuntu

After reboot, launch Ubuntu and install tmux

sudo apt update && sudo apt install tmux -y

Drop SSH sessions directly into WSL — PowerShell (Admin)

Set-Content -Path 'C:\ProgramData\ssh\sshd_config' -Value @'
PubkeyAuthentication yes
PasswordAuthentication no
KbdInteractiveAuthentication no
ForceCommand wsl.exe

Match Group administrators
       AuthorizedKeysFile __PROGRAMDATA__/ssh/administrators_authorized_keys
'@ -Encoding ascii
Restart-Service sshd

After this, new SSH sessions drop straight into WSL.
